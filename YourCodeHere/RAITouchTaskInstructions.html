Replace or modify this view controller

Create a number of views in this view control of size 50x50 that move when you touch them.

  • The views should be added by clicking a button and start at a random location on the screen that does not overlap any other view on the screen.
  • Each time any of the views are touched the view should move to a new location and not overlap any other view on the screen.
  • There should be a number inside each view that shows the number of times that view has been touched.
  • The views bounds should always be within the screen area (both landscape and portrait should be tested).
  • Animate the views to their new location.
  • Give the views some style.
