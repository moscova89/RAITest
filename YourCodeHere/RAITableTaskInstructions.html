Replace or modify this view controller

Read the file states.json from the bundle and parse the json to achive the following:

  • Create a table view that has a row for each state.
  • Each row should simply contain the state name in the order it appears in the json file.
  • Each cell should be clickable and when it's clicked a detail view should be pushed via a navigation controller.
  • The details view should contain the rest of the information in the json file as well as the state's flag.
  • The image for the state flag can be retrieved at the following url: http://www.50states.com/no/0XXf.gif where XX is the state abrivations in lower case.
